Dijkstra's Algorithm Logic

dijkstra_path.cpp 

#include <iostream>
#include <vector>
#include <queue>
#define INF 9999
using namespace std;

// SATELLITE PATHFINDING KERNEL
void dijkstra(int start, int n) {
    vector<int> dist(n, INF);
    priority_queue<pair<int,int>, vector<pair<int,int>>, greater<>> pq;
    
    dist[start] = 0;
    pq.push({0, start});
    
    while(!pq.empty()) {
        int u = pq.top().second;
        pq.pop();
        
        // Check adjacency matrix/list for V
        for(auto i : adj[u]) {
            int v = i.first;
            int weight = i.second;
            
            if(dist[v] > dist[u] + weight) {
                dist[v] = dist[u] + weight;
                pq.push({dist[v], v});
            }
        }
    }
}

Tracing Example

Graph G: A(0)->B(4), A(0)->C(1), C(1)->B(2), B->D(1)

Iteration 1 (Start A):
Dist: [A:0, B:INF, C:INF, D:INF]
Visit A. Update B(4), C(1).
Iteration 2 (Pick C - Min 1):
Dist: [A:0, B:4, C:1, D:INF]
Visit C. Path C->B is 1+2=3. 3 < 4. Update B(3).
Iteration 3 (Pick B - Min 3):
Dist: [A:0, B:3, C:1, D:INF]
Visit B. Path B->D is 3+1=4. Update D(4).

Optimal Path: A -> C -> B -> D

COMPARISON & OBSERVATIONS

Feature Dijkstra Bellman-Ford Floyd-Warshall
Speed (Dense) Fastest Slow Very Slow
Negative Edges Fails Handles Handles
Implementation Moderate Simple Simple
Suitability Best for Drone Path Route Calculation Network Analysis

Observation: For real-time drone navigation where edge weights (distance) are non-negative, Dijkstra's algorithm is superior due to O(V log V) efficiency compared to Bellman-Ford's O(VE).
